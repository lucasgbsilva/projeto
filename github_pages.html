Hospedagem através do GitHub Pages

Descubra como publicar seu próprio site simples no GitHub Pages e aprenda a compartilhar seus projetos de forma fácil

Hospedagem Fácil e Eficiente

O GitHub Pages é uma forma rápida e simples de abrigar páginas estáticas, inclusive projetos em React. É possível hospedar websites diretamente a partir de um repositório no GitHub, facilitando a partilha de projetos.

Para publicar um site em React no GitHub Pages, crie um repositório público para o seu projeto. Em seguida, faça upload dos arquivos de compilação do React (gerados usando npm run build) para o seu repositório. Depois, ative o GitHub Pages nas configurações do repositório, selecionando a pasta build como o diretório principal.

Customizando Nomes de Domínio

Adicionalmente, você tem a opção de personalizar o endereço do seu site para dar uma aparência mais profissional. No GitHub Pages, é viável estabelecer um domínio customizado e realizar as configurações necessárias de DNS para direcioná-lo ao GitHub Pages.

Usando o GitHub Pages, você pode divulgar seu site React para o público de forma simples e sem custos.
